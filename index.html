Frequently Asked Questions

The basic difference between Flex and Grid is that flex is used to design one dimensional layout and grid is used to design two-dimentional layout.A flex layout can also wrap in multiple rows or columns and flexbox treats each row or column as a separate entity, based on its content and the available space. On the other hand, CSS Grid lets you work along two axes: horizontally and vertically

The basic difference between Bootstrap and Tailwind is that Tailwind CSS is not a UI kit. Bootsrap is Component-based which helps you build sites and web applications faster by offering prebuilt web components. On the other side, Tailwind is a utility-first CSS framework for rapidly building custom designs.

In CSS, the term "box model" is used when talking about design and layout. The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.

One of the most important features of HTML5 is its semantics. Semantic HTML refers to syntax that makes the HTML more comprehensible by better defining the different sections and layout of web pages. It makes web pages more informative and adaptable, allowing browsers and search engines to better interpret content. header,footer,article,nav are some examples of semantics.
